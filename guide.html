星詠ノ書の作り方

— AIエージェントで「AI占いツール」を作る道のり —

ご覧いただいた占いツール「星詠ノ書」は、プログラミングの専門知識がなくても、AIエージェント（Claude Code など）に指示するだけで形になります。

「ChatGPTにプロンプトを打って占ってもらう」段階から一歩進んで、“占い体験そのものをツールとして作り、配布・販売できる” ——それがこのページで案内する世界です。以下は、実際に星詠ノ書が生まれるまでの流れです。

— 六つの工程 —

1

企画 — 何を、誰に占うか決める

占術（タロット／西洋占星術／数秘術など）と、届けたい相手・シーンを決めます。星詠ノ書は「恋愛・仕事・金運」を、神秘的な体験とともに“持ち帰れる鑑定書”にする、という設計にしました。

コツ：最初に「どんな成果物が手元に残るか（PDF・画像など）」を決めておくと、売れる形に直結します。
2

AIエージェントに作らせる

Claude Code などのAIエージェントに、企画の内容を伝えて実装してもらいます。「こういう占いツールを、この雰囲気で、こんな流れで」と会話するだけで、画面・ロジック・演出のコードが出来上がっていきます。

伝え方の例：ダークで神秘的な世界観 / カードをめくる演出 / 鑑定書をPDFで出せるように
3

デザインと演出を磨く

世界観づくりが“凄さ”の核心です。配色・フォント・アニメーション（星屑・カードめくり・鑑定文のタイプ表示・音）を重ねて、「おお！」と感じる体験にしていきます。ここもAIと対話しながら細部を詰めます。

星詠ノ書では「墨色の夜空 × アンティークゴールド」に統一し、落款印まで添えて“本物の鑑定書”感を出しました。
4

公開する（固定URL）

完成したツールをネット上に公開して、誰でも開けるURLにします。静的なサイトなら無料で公開でき、URLも固定。SNSやセミナー資料に貼って、その場で体験してもらえます。

配布の仕方次第で「体験期間」を設ける（例：2週間で作り方案内に切り替え）といった導線も組めます。
5

AIを組み込んで“本物の鑑定”に

体験版のロジックを、AI（Claude や ChatGPT の API）による本物の文章生成に差し替えると、一人ひとりに深くパーソナライズされた鑑定が出せるようになります。

重要：APIキーは配布ファイルに置かず、必ず“サーバー”側に隠します。利用回数や期間の管理もサーバーで行うと安全・確実です。
6

届ける・販売する

完成した鑑定（PDF鑑定書・結果画像）は、そのまま“売れる納品物”になります。ココナラ等のスキル販売や、自分の発信からの導線にのせて、体験を価値に変えていきます。

「AIで作った」ことより「どんな体験・成果物が手に入るか」を主役に見せるのが、選ばれるコツです。

— ひとつだけ、大切な注意 —

AIのAPIを組み込む本格版では、APIキーを配布ファイルに絶対に埋め込まないこと。埋め込むと誰でも取り出せてしまい、利用料が想定外に膨らむ危険があります。キーはサーバーに隠し、期間・回数の制限もサーバー側で管理する——これが安全に運用する鉄則です。
