# 星詠ノ書 — HOSHIYOMI AI Fortune Atelier

生年月日と御名前から、**西洋占星術 × 数秘術 × タロット**の三占術を編み合わせ、
神秘的な体験とともに「鑑定書」を綴るWeb占いツールです。

🔮 **公開URL**: https://ryoya9595.github.io/uranai-studio/

## 特徴

- **完全ローカル動作・APIキー不要** — 名前×生年月日からシード乱数で決定論的に鑑定を生成（同じ人は何度でも同じ結果）
- **儀式的な体験フロー** — 入力 → 鑑定中の儀（星座盤が回る）→ タロット三枚を自分でめくる → 鑑定結果
- **三占術ミックス** — 12星座・ライフパスナンバー・大アルカナ22枚
- **持ち帰れる成果物** — A4・3ページの鑑定書PDF（落款印つき）＋ SNSシェア用の結果カード画像
- **演出** — 星屑アニメ、カードフリップ、鑑定文のタイプライター表示、WebAudioの音響（ミュート可）

## 使い方

`index.html` をブラウザで開くだけ。ビルド不要・依存ゼロの静的サイトです。

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面構造（入力／儀式／カード／結果／印刷用鑑定書） |
| `style.css` | 墨色の夜空 × アンティークゴールドの世界観・印刷レイアウト |
| `app.js` | 鑑定エンジン・SVG描画・タロット・PDF/シェア画像生成・音響 |
