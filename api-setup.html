APIの設定

— ご自身の無料AIキーで、星詠ノ書を使い続ける —

体験期間をお楽しみいただき、ありがとうございました。ここから先は、ご自身の無料AIキー（Google Gemini）を設定すると、引き続き星詠ノ書で鑑定を受けられます。

キーはこの端末（ブラウザ）の中だけに保存され、外部には送られません。鑑定はあなたのGoogleアカウントで動きます。Geminiには十分な無料枠があるので、通常のご利用なら費用はかかりません。

— 三つの手順 —

1

Google AI Studio をひらく

aistudio.google.com/apikey を開き、お手持ちのGoogleアカウントでログインします。

2

APIキーを無料で発行する

「Create API key（APIキーを作成）」を押すと、AIza… で始まる文字列が発行されます。これをコピーしてください。

クレジットカードの登録は不要です。無料枠の範囲で鑑定を受けられます。
3

下の欄に貼り付けて保存

コピーしたキーを下の入力欄に貼り付け、「保存して星詠ノ書をひらく」を押すだけ。キーが有効か自動で確認します。

キーを表示　·　保存したキーを削除

— 安全のために —

キーはこの端末のブラウザにのみ保存されます（サーバーには送りません）。共用パソコンでは使い終わったら「保存したキーを削除」を押してください。キーは他人に教えないようご注意ください。
